# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and
personal attacks are not tolerated. We welcome contributors regardless of
background or experience level.

## Reporting

To report unacceptable behavior, contact the maintainer privately via the email
on the maintainer's GitHub profile, or open a GitHub Security Advisory if the
matter is sensitive. Reports are handled confidentially and on a best-effort
basis.
